Hope Gaming


Hope Gaming is a project that I created to practice HTML, CSS and Javascript now that I have finished my classes. This project will help to keep my website development skills fresh while allowing me to try new things and further my skill set. Gaming has always been a passion of mine, and I thought it would be exciting to create a site based on this topic. The plan is to create a simpe functional site first, then add more advanced functionality as I continue.

Hope Gaming is named as a tribute to Rhode Island, the state I was born and raised in. "Hope" is the state's motto. The state was once home to 38 Studios, which created the not so successful game Kingdoms of Amalur. The studio greatly mismanaged the money loaned to them and quickly closed shop after releasing there one and only game. The failure of this company only hurt the states already underwhelming economy.
